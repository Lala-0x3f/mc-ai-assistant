# RELEASE v0.1.4-alpha

## 版本与范围
- 插件版本：0.1.4-alpha
- 主要变更：新增可选的经济扣费支持（Vault/CMI 生态），版本号同步。

## 变更摘要
- 支持在安装 Vault/CMI 等经济插件时，为每次 @ai 调用自动扣费。
- 新增 `economy` 配置段，包含启用开关、单次扣费金额、余额不足与扣费失败提示文案。
- plugin.yml 软依赖加入 Vault、CMI，版本号同步为 0.1.4-alpha；POM 增加 VaultAPI（provided）。
- README、示例配置补充了经济扣费说明。

## 升级指南
1. 替换构建后的插件 Jar（或拉取代码后 `mvn clean package`）。
2. 在已有配置中新增 `economy` 段，按需开启并设置 `cost_per_use`、提示文案。
3. 确认服务器已安装 Vault 或 CMI（并提供经济接口），否则扣费自动跳过但会记录日志。
4. 重启服务器或重载插件生效。

## 已知注意事项
- 扣费发生在发送 AI 请求前，余额不足或扣费失败会直接提示玩家并取消响应。
- 经济调用在主线程执行，异步聊天事件会自动切回主线程，避免线程不安全。需确保经济插件响应正常。*** End Patch

## ✨ 主要功能

### 🤖 AI 对话功能
- **命令**: `@ai [消息]`
- **功能**: 与 AI 进行智能对话
- **特性**: 支持上下文理解、多轮对话

### 🔍 网络搜索功能
- **命令**: `@search [查询]`
- **功能**: 实时网络搜索并返回结果
- **特性**: 智能搜索、结果摘要

### 🔗 可点击链接支持
- **功能**: 搜索结果中的链接可直接点击打开
- **特性**: 
  - 蓝色下划线样式

### 📊 速率限制管理
- **功能**: 防止 API 滥用

## 🐛 重要修复

### 1. API 响应压缩解析问题
- **问题**: 手动设置 `Accept-Encoding` 导致响应无法解析
- **修复**: 移除手动设置，让 OkHttp 自动处理压缩
- **影响**: `@ai` 和 `@search` 功能都受益

### 2. 搜索结果控制台显示问题
- **问题**: 搜索结果只在客户端显示，控制台看不到
- **修复**: 添加控制台广播功能
- **影响**: 管理员可以在控制台查看搜索结果

### 3. 链接处理和显示优化
- **问题**: markdown 链接格式影响阅读体验
- **修复**: 自动提取链接并创建可点击组件
- **影响**: 更好的用户体验和视觉效果

## 🚀 部署说明

### 1. 编译
```bash
mvn clean package -DskipTests
```

### 2. 安装
- 将 `target/mc-ai-assistant-0.0.3.jar` 复制到服务器 `plugins` 目录
- 重启服务器或使用 `/reload` 命令

### 3. 配置
- 编辑 `plugins/McAiAssistant/config.yml`
- 设置正确的 API URL 和密钥
- 根据需要调整其他配置项

### 4. 测试
- 使用 `@ai 你好` 测试 AI 对话功能
- 使用 `@search 天气` 测试搜索功能
